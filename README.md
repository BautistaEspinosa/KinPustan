# KinPustan
De lo tradicional a lo digital, sin perder el alma.

KinPustan es una aplicación pensada para modernizar la gestión de negocios pequeños y medianos, llevando la eficiencia digital a un entorno tradicional sin perder la esencia que hace únicos a los comercios locales.

# 🌱 Historia
Inspirada en el legado de un negocio familiar, KinPustan es una herramienta creada para apoyar a las pequeñas tiendas, como la que comenzó un padre en su pueblo natal. Con un enfoque en facilitar la gestión de inventarios, ventas y más, esta aplicación combina tecnología moderna con el corazón de la tradición.

# 🛠️ Características
Gestión de inventarios: Organiza tus productos, controla existencias y mejora la rotación de tu inventario.

Ventas simplificadas: Realiza ventas rápidas, con un sistema intuitivo y fácil de usar.

Informes detallados: Genera reportes de ventas, ingresos y más para tomar decisiones informadas.

Interfaz amigable: Diseñada pensando en la simplicidad, para que todos puedan usarla con facilidad.

Accesibilidad desde cualquier dispositivo: Ideal para emprendedores con acceso limitado a tecnología avanzada.

# ⚙️ Tecnologías
Backend: Java 17, Spring Boot

Frontend: React.js (¡Para que puedas aprender y explorar nuevas tecnologías!)

Base de Datos: Oracle (H2 usado temporalmente)

Contenerización: Docker

Otros: API REST, Spring Cloud Config
